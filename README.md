#  AI Assistant Core

This is my Project #4 in my AI learning journey.

It is a simple AI assistant system built in Python that demonstrates how memory-based assistants work.

---

##  What it does

- Talks to the user (basic chatbot system)
- Stores conversations in memory (JSON file)
- Searches previous conversations
- Uses stored memory to improve responses

---

##  System Structure

- `assistant.py` → Main brain (handles conversation flow)
- `memory.py` → Handles saving and loading memory
- `search.py` → Finds relevant past messages
- `memory.json` → Stores all conversations

---

##  How it works

User message → Search memory → Generate response → Save conversation → Reply
